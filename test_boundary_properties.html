边界设置移动测试

测试步骤：

  1. 启动应用
  2. 选择或创建一个项目
  3. 选择或创建一个画图
  4. 检查右侧属性面板是否显示边界设置
  5. 尝试修改边界值并应用
  6. 验证边界预览功能

预期结果：

  • 选中画图后，右侧属性面板显示"当前画图已选择，可在下方设置操作边界"
  • 边界设置区域显示在属性面板底部
  • 边界设置包含位置（X, Y）和尺寸（宽, 高）输入框
  • "应用边界"和"预览边界"按钮正常工作
  • 左侧画图管理器不再显示边界设置

UI改进：

  • 边界设置区域有清晰的分隔线
  • 输入框有合适的标签和样式
  • 按钮排列美观
  • 整体布局符合设计规范
